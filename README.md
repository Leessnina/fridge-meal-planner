# 🥘 食材盘点器 (Fridge Meal Planner)

> 打开冰箱，3 秒决定今晚吃什么！

一个移动端优先的单页 Web 应用，根据冰箱剩余食材智能推荐晚餐菜谱。无需安装，双击 `index.html` 即可使用。

## ✨ 功能

- 🥬 **食材盘点**：勾选冰箱中的蔬菜、肉类、主食，支持自定义食材
- 🧂 **调料库存**：记录家中现有调料，自动记忆
- 🔄 **调料平替**：智能推荐可替代的调料方案（如蚝油→生抽+糖）
- ⚠️ **略蔫优先**：优先推荐消耗临期食材的菜品
- 🚫 **忌口过滤**：排除不吃的食材
- 📝 **自定义菜谱**：添加私房菜谱，与内置菜谱一起参与匹配
- 📊 **匹配评分**：综合匹配率、食材状态、调料完整度等智能排序

## 🚀 使用方法

1. 下载 `index.html` 和 `recipes.js` 放在同一目录
2. 双击 `index.html` 在浏览器中打开
3. 勾选冰箱里有的食材和调料
4. 点击「🔍 推荐菜谱」查看结果

## 📁 文件结构

```
fridge-meal-planner/
├── index.html      # 主应用（UI + 匹配引擎）
├── recipes.js      # 菜谱数据库（35个内置菜谱 + 自定义加载）
└── README.md       # 项目说明
```

## 🛠 技术栈

- 纯前端：HTML + CSS + Vanilla JavaScript
- 无构建工具、无框架、无外部依赖
- 通过 file:// 协议直接运行
- localStorage 持久化用户数据

## 📝 如何添加新菜谱

**方法一：应用内添加**
1. 打开 `index.html` → 底部「📝 自定义菜谱管理」
2. 填写表单提交

**方法二：编辑 recipes.js**
1. 按文件顶部的模板格式添加菜谱对象
2. 保存后刷新页面即可
